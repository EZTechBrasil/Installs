ATENÇÃO !  

Antes de iniciar esta instalação é necessário: 

1. Se certificar que o .Net 3.5 já está instalado. 
   Caso não esteja, o processo de intalação será cancelado. 
   Para instalar, acesse a área de downloads no site da EZTech.

2. Ter os drives já instalados. Se não tiver certeza, execute os procedimentos no manual de instalação

3. Conectar a unidade EZforecout a uma porta usb do computador e ligar.
