ATENCION!

Antes de iniciar esta instalación es necesario hacer lo siguiente:

1. Asegúrese Net 3.5 ya está instalado
   Caso de que esté instalado no, se cancelará el proceso de instalación.
   Para instalar, acceder al área de descarga en la página web EZTech.

2. Tener las unidades de dispositivos ya instalados. Si no está seguro, consulte el manual de instalación.

3. Conecte la unidad EZForecourt a un puerto USB y vuelva a encenderlo.
