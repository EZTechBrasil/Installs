ATTENTION ! 

Before you start this installation it is necessary to do the following:

1. Ensure .Net 3.5 is already installed
   If it´s not installed, the installation process will be cancelled.
   To install, access the download area on the EZTech website.

2. Have the device drives already installed. If you are not sure, refer to the installation manual.

3. Connect the EZForecourt unit to an USB port and turn it on.
